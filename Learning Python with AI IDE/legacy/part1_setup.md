## 1. 第一步：验明正身 (System Check)

在下载任何软件之前，我们需要先搞清楚一个问题：**你的电脑到底是谁？**
特别是使用 Mac 的同学，你的芯片类型决定了你要下载哪个版本的“魔杖”。

**[画面演示：Mac 屏幕录屏]**

如果你是 **Mac 用户**：
请看屏幕左上角，有一个苹果图标 ()。点击它，选择 **"关于本机" (About This Mac)**。
请注意看 **"芯片" (Chip)** 这一行：
*   如果写着 **Apple M1, M2, M3** 或者 M4：恭喜，你是 **Apple Silicon** 架构。记住这个词。
*   如果写着 **Intel**：那你就是 **Intel** 架构。

如果你是 **Windows 用户**：
大部分现代电脑都是 **x64** 架构，只要你的系统是 64 位的通常就没问题。

---

## 2. 第二步：分发魔杖 (安装 Miniconda)

搞清楚了身份，我们就要去领取我们的魔法棒了——它的名字叫 **Conda**。
我们推荐使用轻量级的 **Miniconda**。

大家可以前往 Miniconda 的官网，或者国内的清华源。
*   Mac M 芯片的同学，请下载 **Apple M1 64-bit pkg**。
*   Windows 同学，下载 **Windows 64-bit exe**。

下载好之后，双击安装。
这里有一个**关键点**需要提醒 Windows 用户：
安装安装程序可能会问你：*"Add Miniconda3 to my PATH environment variable?"*
虽然它标了红色说不推荐，但对于新手，我**建议勾选**。这样你以后在黑框框里打字会方便很多。

**[音效：安装完成的提示音]**

安装好了吗？我们来验证一下。
请打开你的 **终端 (Terminal)**。
*   Mac 用户按 `Cmd + Space` 搜索 "Terminal"。
*   Windows 用户搜 "PowerShell"。

在里面输入：`conda --version`，然后回车。
如果你看到了一串版本号，比如 `conda 23.x.x`，那就说明——魔杖已就位。

---

## 3. 第三步：开辟空间 (创建虚拟环境)

现在，我们要用这根魔杖，在你的电脑里变出一个**“AI 课程专用房间”**。

请在终端里，跟着我输入这行咒语：
`conda create -n ai_course python=3.12 -y`

这行命令的意思是：
*   `create`：造一个房间。
*   `-n ai_course`：房间名叫 ai_course。
*   `python=3.12`：房间里必须摆放 Python 3.12 版本的家具。

输入完回车，你会看到一堆进度条在跑。稍等片刻。

当屏幕停下来，提示你 "To activate this environment..." 的时候，就是房间造好了。，
现在，让我们**走进这个房间**。输入：
`conda activate ai_course`

**大家注意看！**
你的命令行最前面，是不是多了一个小括号，写着 `(ai_course)`？
只要这个括号在，就说明你现在是安全的。你所有的操作，都被隔离在这个小房间里了。

---

## 4. 第四步：装修驾驶舱 (IDE 设置)

最后一步，我们把 **Antigravity IDE** 请出来。虽然我们装好了 Python，但 IDE 还不知道它的位置。我们需要把它们**连接**起来。

**[画面演示：打开 IDE]**

首先，请安装两个必备插件：
1.  **Python 插件**：这是必须的。
2.  **Chinese 中文插件**：让界面更友好。

接下来是**最最重要**的一步，很多同学就是在这里栽了跟头。
我们需要告诉 IDE：“嘿，用我刚才造的那个 `ai_course` 房间，别用系统自带的。”

请按下快捷键打开命令面板（Mac 是 `Cmd+Shift+P`，Win 是 `Ctrl+Shift+P`）。
输入 `Select Interpreter`（选择解释器）。

在弹出的列表里，**一定要仔细找**。
找到那个写着 `ai_course` 的选项。它可能长得像 `Python 3.12.x ('ai_course': conda)`。
点击它！

看看 IDE 的**右下角**，是不是显示了 `('ai_course')`？
如果是，恭喜你。
**Antigravity 飞船的引擎，已经成功对接到了你的 AI 专用燃料舱。**
