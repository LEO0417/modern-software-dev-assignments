API Reference / API 参考文档

Base URL / 基础 URL: https://api.example.com/v1

Authentication / 认证:
  Provide header X-API-Key: <your key>
  提供头部 X-API-Key: <你的密钥>

Endpoints / 端点:
  GET /users/{id}
    - Returns 200 with JSON: {"id": <string>, "name": <string>}
    - 返回 200 和 JSON: {"id": <字符串>, "name": <字符串>}
