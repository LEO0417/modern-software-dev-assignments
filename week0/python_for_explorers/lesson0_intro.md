## 0. 开场 (Opening) - 3 min

**管理员**：
欢迎来到数据档案馆，Explorers (探险家们)。

从今天开始，我们的任务不再是“飞”，而是“挖掘”。

---

## 1. 我们的工具 (The Tools) - 5 min

**管理员**：
在 Anti-gravity IDE 强大的文件管理系统下，探险家需要处理三种主要的“矿石”：

1.  **Text Files (文本文件)**：
    这是最基础的记录（`.txt`, `.log`）。在斯坦福课程中，你的 AI 可能会阅读几千个这样的文件。我们要学会用 Python 打开它，读懂它。

2.  **Structured Data (结构化数据 - JSON)**：
    还记得 API 返回的字典吗？如果我们要把它存到硬盘上，它就变成了 `.json` 文件。这是现代软件配置的标准格式。

3.  **Context Managers (上下文管理器)**：
    这是一个高级魔法——`with open(...)`。
    Anti-gravity 的 AI 副驾驶会告诉你：这是处理资源最安全、最优雅的方式。

---

## 2. 核心任务：持久化 (The Mission) - 5 min

**管理员**：
在这一周，我们的目标是让程序拥有**记忆**。

我们不仅要读取数据，还要分析数据。
我们将编写脚本来扫描成千上万行的日志文件，找出究竟是哪一次请求导致了系统崩溃。
这叫 **Log Analysis (日志分析)**。
这是每一位要去 Stanford Course 的指挥官必须掌握的侦查技能。
