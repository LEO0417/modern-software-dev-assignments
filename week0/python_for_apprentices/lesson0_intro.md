## 1. 解锁右翼：The AI Co-pilot - 5 min

**领航员**：
(手势指向屏幕右侧)
请把目光移到屏幕的最右侧。
在 Infants 阶段，那个区域可能是折叠的，或者是黑色的。
但现在，请点开它。

看到了吗？那里有一个对话框。
这不仅仅是一个聊天窗口，它不是一个网页版的 ChatGPT，也不是一个用来陪聊的机器人。
它是 **Google Anti-gravity IDE** 的灵魂核心。

我们要称呼它为：**Right Panel (右舱)**。
它和左边的文件、中间的代码是**完全打通**的。

*   **Left Panel (左舱)**：是你的记忆（Files）。
*   **Center Panel (主舱)**：是你的双手（Editor）。
*   **Right Panel (右舱)**：是你的**外脑 (Exocortex)**。

在传统的编程时代（比如用 VS Code 的旧时代），程序员遇到不会写的代码，需要：
切出去 -> 打开浏览器 -> 搜 Google/StackOverflow -> 复制 -> 切回来 -> 粘贴。
这叫 **Context Switch (上下文切换)**，是效率的杀手。

但在 Anti-gravity 里，你就在右舱输入：
> "帮我解释一下左边代码里的这个函数做了什么？"
它会立刻读取你中间打开的文件，给出极其精准的答案。
这就是 **AI-Native** 的力量。

## 2. 我们的任务 (The Mission) - 4 min

**领航员**：
在这个 "Apprentice" 阶段，我们要通过这个强大的右舱，来重新认识 Python。
我们要掌握三把开启斯坦福课程大门的金钥匙：

1.  **Variables & Types (变量与类型)**：
    学会精准地定义“这是什么”。
2.  **Functions (函数)**：
    学会封装逻辑，制造零件。
3.  **Data Structures (数据结构)**：
    学会管理成百上千条数据。
