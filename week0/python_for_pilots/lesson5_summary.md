## 1. 晋升测试 (Promotion Test) - 5 min

**领航员**：
在你获得 Explorer (探险家) 徽章之前，思考这几个问题：

1.  如果在爬虫代码里，你忘记写 `import requests`，会发生什么？
    *(答案：NameError，因为你没带工具)*
2.  如果你写了 `pip install pandas` 但代码里用的是 `import numpy`，能行吗？
    *(答案：不行，你买的是锤子，却想用螺丝刀)*
3.  如果有一个 API 每秒钟只能访问一次，你的循环里应该加什么？
    *(答案：time.sleep)*

这些不仅是代码问题，更是工程常识。
